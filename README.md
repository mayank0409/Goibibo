# Goibibo
Project for PyCon 2016 - Goibibo

1. Clone the repository - ```git clone https://github.com/mayank0409/Goibibo.git ```
2. Open the terminal and move into the cloned folder ('Goibibo') - ``` cd Goibibo ```
3. ``` npm install ```
4. ``` bower install ```
5. Run the server - ``` node app.js ```
6. Now the server will be accessible at http://localhost:8080
